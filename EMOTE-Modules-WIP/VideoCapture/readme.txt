First you must download Divx codec as it uses it for compression
The required libs are in the lib folder.
You must also download the Emgu CV and inside it to the default path.Once installed copy the x86 and x64 folders from the installed path to the bin folder of the camera project.
You can download the openCV from http://sourceforge.net/projects/emgucv/
Install it and then restart the computer


This is a generic HD camera module so you can use as many as you want. Just use a different module ID from the app.
It doenst record audio, just video
